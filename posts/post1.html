A Sample Post: Math, Figures, and Typography

April 17, 2026

This is a minimal, static “paper-like” blog template inspired by colah.github.io: centered column, generous margins, Computer Modern typography, and KaTeX math — no backend, no build tools.

Inline Math

KaTeX renders inline math delimited by $...$, for example Euler’s identity $e^{i\pi}+1=0$ and a Gaussian $e^{-x^2}$.

Displayed Equations

Displayed equations use $$...$$:

$$\int_{-\infty}^{\infty} e^{-x^2}\,dx = \sqrt{\pi}$$

And a classic series: $$\sum_{n=1}^{\infty}\frac{1}{n^2}=\frac{\pi^2}{6}$$

Figures (Academic Style)

Figures use semantic <figure> and <figcaption>, with a subtle border and centered caption.

Headings, Code, and Spacing

The layout targets readability: max width around 680px, line-height around 1.7, and restrained color. 

/* Example code block */
body {
  background: #fdfdfd;
  color: #222;
  line-height: 1.7;
}
A good rule of thumb: make it easy to read, and hard to get lost.
